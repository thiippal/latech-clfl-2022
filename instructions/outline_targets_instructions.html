How to perform the task

  1. Read the instructions below and make sure you understand them.
  2. Find the target(s) of the diagram described by the outlined text. Use the zoom tool to make sure you see the whole diagram if needed.
  3. Use the polygon tool (hotkey: w) or the rectangle tool (hotkey: e) to draw accurate outlines for the target(s). Use the label 'Target' for the outline.
  4. Click Submit to submit the completed assignments.

Payment information: You will be paid $0.3 per assignment. You will be paid immediately after your work has been verified by other users.

Purpose statement: These tasks are part of a research project conducted at the University of Helsinki.

Instructions

You will be shown outlines for diagram elements that contain text, letters or numbers.

You must draw accurate outlines for the target(s) that the text refers to.
